# 项目简介

本系统为基于深度学习的水面漂浮垃圾检测识别系统，采用最新的yolov8算法，通过Python代码实现。系统包含友好的UI界面，便于用户操作。主要功能模块为对水面漂浮垃圾进行高效检测与识别，通过训练的数据集，实现对不同种类垃圾的精准分类。技术特点包括深度学习模型的高准确度、实时性检测，以及对复杂场景的适应能力。


## 核心功能
- 基于YOLOv8算法实现水面漂浮垃圾检测与识别
- 提供友好的用户界面(UI)，简化操作流程
- 通过训练数据集对不同种类垃圾进行精准分类
- 支持实时检测功能，提高效率
- 适应复杂的水面检测场景

## 适用场景
- 水环境保护监测
- 河流、湖泊垃圾污染评估
- 水面清洁作业指导
- 环境保护研究
- 水质监测辅助系统

## 技术特点
- 采用Python编程语言开发
- 集成最新深度学习技术
- 高准确度的垃圾识别能力
- 实时检测性能优化
- 良好的场景适应性
